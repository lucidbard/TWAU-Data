These files contain the initial annotation of a video playthrough. The video itself can be found on YouTube at https://www.youtube.com/playlist?list=PLYH8WvNV1YEmYe_pKdwLxBrdpi2M-G5ni.

The reconstructed ink file from the video playthrough is TWAU_001.ink. It can be played using the open source INK authoring tool or player which can be found here: https://github.com/inkle/ink

The first episode's SIG encoding, mapped to the transcript embedded in
the VGL file was generated by the ink playthrough using the following
decisions): TWAU_001.vgl.

Scheherazade can be downloaded at http://www.cs.columbia.edu/~delson/software.shtml.

Also included is the web file and javascript program used to generate the transcript.
